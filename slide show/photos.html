Bersenang-senang dengan foto Anda

Shotwell adalah pengelola foto praktis yang siap untuk gadget Anda. Hubungkan kamera atau telepon untuk mentransfer foto Anda, maka mudah untuk membagikannya dan menjaganya tetap aman. Jika Anda merasa kreatif, Anda dapat mencoba banyak aplikasi foto dari Ubuntu Software Center.

Included software

  • Shotwell Photo Manager

Supported software

  • GIMP Image Editor

  • Pitivi Video Editor
